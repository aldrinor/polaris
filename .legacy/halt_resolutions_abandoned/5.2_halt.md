## Why pre-emptive halt (cost saving)

If we attempt 5.2 now, Codex would emit REQUEST_CHANGES for substrate gaps
in every Phase 1+ feature (the same root cause: cluster not online). Each
P0/P1 finding-batch costs ~$0.10 + 5 min wall-clock × 50+ findings × 3
iterations to halt-condition #4 = ~$15-20 + 4-6 hours wall-clock. That's
all rediscovery of the same single blocker.

Same reasoning applied to walkthrough prep packs (`walkthrough_prep_packs_structural_halt.md`)
saved ~$1.20 + 30 min there.
