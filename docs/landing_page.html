The Solution

8-Node Verified Research Pipeline

POLARIS processes your research query through an 8-node pipeline that searches, interviews experts, extracts evidence, verifies every claim, and synthesizes a comprehensive report with full bibliographic provenance.

1 Plan 50 sub-queries from 9 expert perspectives
2 Search 6 engines, federated + agentic loop
3 STORM 5 expert interviews, 3 rounds each
4 Analyze Fetch pages, extract atomic facts
5 Verify NLI per claim, cross-source check
6 Evaluate Gap analysis, iterate if needed
7 Synthesize Cluster, outline, cite, assemble
8 Report 12,000 words, 200+ citations

NLI Claim Verification

Every extracted fact verified against source content using MiniCheck (flan-t5-large) Natural Language Inference. Machine-verifiable verdicts: SUPPORTED or NOT_SUPPORTED with probability scores.

STORM Multi-Perspective

Stanford STORM methodology: 5 expert personas interview the research topic from different professional angles. Ensures comprehensive coverage that single-perspective tools miss.

5-Signal Evidence Scoring

Every evidence item scored across 5 dimensions: relevance, authority, density, freshness, and grounding. GOLD/SILVER/BRONZE tier classification ensures only high-quality evidence enters the report.

Self-Correcting Pipeline

Pipeline iterates up to 5 times, re-searching and re-verifying until quality gates pass. Gap analysis identifies missing topics. Convergence detection prevents infinite loops.

8-Layer Audit Trail

Every claim traceable from report sentence through citation, evidence item, verification verdict, NLI score, source URL, search query, back to the STORM perspective that generated the query.

Agentic Search Loop

Search-read-reason-refine cycle across 8 rounds discovers sources that keyword search misses. 6 search engines federated with neural, academic, and web coverage.

Cloud to Sovereign: Change One URL

POLARIS uses OpenAI-compatible APIs throughout. Switch from cloud to sovereign by changing 3 environment variables. No code changes. No retraining. The same pipeline, the same quality.

# ========== CLOUD MODE ==========
OPENROUTER_API_KEY=sk-or-v1-your-key
OPENROUTER_BASE_URL=https://openrouter.ai/api/v1
OPENROUTER_DEFAULT_MODEL=moonshotai/kimi-k2.5

      ↓   Change these 3 lines   ↓

# ========== SOVEREIGN MODE ==========
OPENROUTER_API_KEY=not-needed
OPENROUTER_BASE_URL=http://localhost:8080/v1
OPENROUTER_DEFAULT_MODEL=Qwen/Qwen2.5-32B-Instruct

Cloud Mode

Use OpenRouter for LLM inference (400+ models), Serper for web search, Exa for neural search, Jina for content extraction. Best for teams evaluating POLARIS or without GPU infrastructure.

Sovereign Mode

Use vLLM for local inference, SearxNG for self-hosted search, Crawl4AI for content extraction. Zero data egress. Air-gapped capable. Required for classified, regulated, and sensitive research.
