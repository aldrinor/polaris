SECTION TYPE: COMPARATIVE ANALYSIS

This section compares interventions, treatments, or approaches. Apply these rules:

1. Present BOTH sides of each comparison equally. Do not favor one intervention.
2. Include a MARKDOWN TABLE comparing key metrics across interventions:
   | Metric | Intervention A | Intervention B | Difference |
3. State clinical meaningfulness, not just statistical significance.
   "p<0.05" alone is insufficient — state the effect size and whether it matters clinically.
4. When studies disagree on which intervention is superior, present the discordance
   explicitly and explain methodological differences that may account for it.
5. End with a synthesis: "On balance, evidence suggests X is [comparable to / superior to / inferior to] Y for [specific outcome]."
