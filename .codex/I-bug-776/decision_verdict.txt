**Decision**

Use **b1 now**, with **b2 as the required general-product follow-up**. Do **not** use adequacy relaxation `(a)` to close this incident.

Rationale: the clinical-safety-preserving fix is to retrieve the missing pivotal primary trials, not lower the evidence bar. For a “current guidelines” NVAF anticoagulation answer, the faithful corpus includes both guideline/synthesis sources and the landmark DOAC trials.

**b1 Scope**

Add `per_query_primary_trial_anchors` only for the curated demo slugs currently missing anchors:

- `afib`
- `dd_novo`
- `dd_lilly`

For `afib`, use compound, disambiguated anchors:

- `ARISTOTLE apixaban`
- `ROCKET AF rivaroxaban`
- `RE-LY dabigatran`
- `ENGAGE AF-TIMI 48 edoxaban`

For `dd_novo` and `dd_lilly`, anchors must be tied to the exact molecule + indication in the vector metadata. No company-wide trial-family stuffing.

**Guardrails**

- Slug-scoped only: no global fallback anchors.
- No bare ambiguous acronyms: use trial + drug/condition where needed.
- Count as T1 only if the retrieved source is a primary trial, registry, or qualifying PubMed primary source.
- Deduplicate by PMID/DOI/NCT.
- Log which anchor produced each counted source.
- If an anchor retrieves mostly reviews/guidelines/off-topic trials, remove it rather than relying on classifier filtering.

**Adequacy Calibration**

Do not change `clinical.json` for this fix. No blanket or guideline-type T1 relaxation.

If `(a)` is designed later, it must be a separate question-type-gated change for explicit guideline/synthesis tasks, with hard exclusions for safety, efficacy, comparative effectiveness, dosing, adverse-event, and intervention-decision questions. It should still require a strong guideline/SR corpus and at least some primary/regulatory support. It should not be used to pass this benchmark.
