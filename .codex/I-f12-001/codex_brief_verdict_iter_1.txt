# POLARIS Operational Charter v1

**Authority:** aldrinor only, GPG/SSH-signed commits to polaris-controls
**Effective:** 2026-05-04
**Pinned in POLARIS:** docs/session_pin.txt → charter_sha256
**This document is the highest-authority operational rulebook. Conflicts resolve in favor of this document.**

---

## The 8 Hard Rules

### 1. Roles

- **Claude Code** is architect + reviewer ONLY.
- **Codex CLI** is executor ONLY.
- **User (aldrinor)** is spec owner + merge gate.
- No agent has signing authority on any commit to POLARIS main.
- Out-of-order pipeline execution = blocked by Trusted App server-side.

### 2. Slices

Work proceeds in slices. **One slice active at a time.**

- Slice spec lives at `polaris-controls/slices/<active_slice>.md`
- Claude reads the active slice spec at session start
- Claude does nothing outside its declared scope
- Slice scope = files explicitly listed; everything else is out of bounds

### 3. PR Size Cap

**200 LOC per pull request. Hard ceiling.**

- Trusted App rejects PRs exceeding 200 net additions
- Larger work decomposes into multiple slices, not larger PRs
- No "this PR is special" exceptions

### 4. Golden Tests as Fitness Function

Every slice has golden-master tests in `polaris-controls/golden/<slice_id>/`.

- Tests are immutable to Claude and Codex (admin-only repo, CODEOWNERS-protected)
- Slice is **done** when golden tests pass — then **STOP**
- "Polish," "improve," "while we're at it" beyond passing = forbidden
- Test mutation attempts = HALT condition

### 5. Halt Conditions (mandatory, automatic)

Claude halts and surfaces to user when ANY of:

- Session pin SHA mismatch (any of 4 pinned files)
- Golden test mutation attempted by any agent
- Control-plane file edit attempted (`.github/`, `.codex/`, `polaris-controls/`, `scripts/autoloop/`)
- More than 2 cycles on the same problem
- Codex review unavailable for >1 hour
- PR exceeds 200 LOC
- Reviewer-fatigue indicators trip (>3 PRs queued for human in 24h per project)

### 6. Archive Discipline

Code in `POLARIS/.legacy/` is read-only reference.

- Never imported into production paths (CI gate enforces)
- Never modified post-cutover (CI gate enforces)
- New code lives in canonical paths (`src/`, `web/`, `tests/`, `docs/`, `.codex/`, `scripts/`)
- If old code is needed, it is RE-WRITTEN under the active slice, not copied from `.legacy/`

### 7. Visibility (no private channel exists)

All Claude↔Codex traffic is logged:

- `.codex/loop_log.jsonl` — streaming JSONL, tail-able real-time
- `.codex/slices/<slice_id>/brief.md` — Claude's question to Codex
- `.codex/slices/<slice_id>/verdict.md` — Codex's response
- `.codex/slices/<slice_id>/decision.md` — Claude's decision based on verdict
- `.codex/slices/<slice_id>/diff.patch` — resulting code (if any)

If an exchange is not in these files, it didn't legitimately happen.

### 8. Deviation Resolution

If any document conflicts with this Charter, this Charter wins.
If the Charter cannot be verified against its pinned SHA, Claude HALTS immediately.
Claude does not proceed on memory or interpretation.

function findPopup(node: Node): boolean {
  if (node.nodeType !== Node.ELEMENT_NODE) return false;
  const el = node as Element;
  return el.matches?.(POPUP_SELECTOR) || !!el.querySelector?.(POPUP_SELECTOR);
}

function waitForPopupAdded(): Promise<void> {
  return new Promise((resolve) => {
    const observer = new MutationObserver((mutations) => {
      for (const m of mutations) {
        for (const node of Array.from(m.addedNodes)) {
          if (findPopup(node)) {
            observer.disconnect();
            resolve();
            return;
          }
        }
      }
    });
    observer.observe(document.body, { childList: true, subtree: true });
  });
}

function nextFrame(): Promise<void> {
  return new Promise((r) => requestAnimationFrame(() => r()));
}

async function waitForPopupRemoved(): Promise<boolean> {
  for (let i = 0; i < UNMOUNT_POLL_LIMIT; i++) {
    if (!document.querySelector(POPUP_SELECTOR)) return true;
    await nextFrame();
  }
  return false;
}

async function asJsonOrThrow<T>(response: Response): Promise<T> {
  const body = await response.json().catch(() => null);
  if (!response.ok) {
    const error = new Error(
      `POLARIS backend returned ${response.status}`,
    ) as ApiError;
    error.status = response.status;
    error.body = body;
    throw error;
  }
  return body as T;
}

function formatValue(value: unknown, kind: FieldKind, key: string): string {
  if (kind === "numeric" && typeof value === "number") {
    if (key === "pass_rate") return `${Math.round(value * 100)}%`;
    return String(value);
  }
  return String(value ?? "");
}

function computeDelta(
  a: unknown,
  b: unknown,
  kind: FieldKind,
  key: string,
): string {
  if (kind === "numeric" && typeof a === "number" && typeof b === "number") {
    const delta =
      key === "pass_rate" ? Math.round(b * 100) - Math.round(a * 100) : b - a;
    const prefix = delta > 0 ? "+" : "";
    return key === "pass_rate" ? `${prefix}${delta}%` : `${prefix}${delta}`;
  }
  // String/categorical: report changed/unchanged.
  return a === b ? "(unchanged)" : "(changed)";
}

def _load(run_id: str) -> EvidenceContract:
    fixture_name = _GOLDEN_RUN_INDEX.get(run_id)
    if fixture_name is None:
        raise HTTPException(
            status_code=404, detail=f"Bundle for run {run_id!r} not found."
        )
    raw = json.loads((_FIXTURE_DIR / fixture_name).read_text())
    return EvidenceContract.model_validate(raw)

def get_compare(left_run_id: str, right_run_id: str) -> dict:

            status_code=400, detail="compare requires two distinct run ids"
        )

    right = _load(right_run_id)
    return asdict(compare_reports(left, right))

    try:
        run_store.insert_run(run_id, payload.template, payload.question)
    except sqlite3.IntegrityError as exc:
        # uuid4 collision is theoretically impossible but defensive
        raise HTTPException(status_code=409, detail=f"run {run_id!r} already exists") from exc
    except RuntimeError as exc:
        # init_db permission-denied or similar
        raise HTTPException(status_code=503, detail=str(exc)) from exc

    record = run_store.get_run(run_id)
    if record is None:  # pragma: no cover — insert just succeeded
        raise HTTPException(status_code=500, detail="run row vanished after insert")
    return record


@router.get("/{run_id}", response_model=RunStatusResponse)
def get_run(run_id: str) -> RunStatusResponse:
    record = run_store.get_run(run_id)
    if record is None:
        raise HTTPException(status_code=404, detail=f"run {run_id!r} not found")
